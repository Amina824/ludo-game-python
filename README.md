# 🎲 Ludo Game in Python

This project is a Python implementation of the classic **Ludo board game**, developed using a Jupyter Notebook. It simulates core game mechanics including dice rolling, player movement, and turn-based gameplay.

## 🚀 Features

* 🎲 Dice rolling simulation
* 👥 Multiple player support
* 🔄 Turn-based gameplay
* 📍 Token movement logic
* 🧠 Basic game rules implementation

## 🛠️ Technologies Used

* Python
* Jupyter Notebook

## 📁 File Structure

* `Ludo.ipynb` – Main notebook containing the game implementation

## ▶️ How to Run

1. Install Jupyter Notebook (if not already installed):

   ```bash
   pip install notebook
   ```

2. Open the notebook:

   ```bash
   jupyter notebook
   ```

3. Run all cells to start the game.

## 📌 Notes

* This is a simplified version of Ludo.
* The game runs inside a notebook environment.
* No GUI is used (text-based logic).
